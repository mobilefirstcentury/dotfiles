CMUS MANUEL
===========
Cmus view
----------
1 : View tree
2 : View sorted
3 : View play­list
4 : View queue
5 : View browser
6 : View filters
7 : View settings

Cmus control
------------
x : Player play
c : Player pause
v : Player stop
b : Player next
z : Player prev

Cmus other
----------
u : update­-cache
i : toggle show_h­idden
^ : L refresh

Cmus window
-----------
g, home : Goto top of the current window.
G, end  : Goto bottom of the current window.
space   : Expand albums in library view (1).
enter   : In views 1-3 start playing the selected track.
tab     : Activate next window. Only relevant in view 1.
k, up   : Goto up one row in the current window.
down, j : Goto down one row in the current window.

Cmus vol
---------
=, + : vol +10%
-    : vol -10%
]    : vol +0 +1
[    : vol +1 +0
}    : vol -0 -1
{    : vol -1 -0

Cmus seek
---------
.        : seek +1m
l, right : seek +5
         : , seek -1m
h, left  : seek -5
